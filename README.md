# 💼 Personal Portfolio Website

A responsive personal portfolio website built using **HTML, CSS, and JavaScript** as part of the **CodeAlpha Frontend Development Internship**.

The portfolio showcases my skills, projects, education, certifications, resume, and contact information with a clean, modern, and responsive user interface.
